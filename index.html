opening script tag type = "text/javascript"> var money; var name; closing script tag

Variable_scope

The scope of a variable is the region of your program in which it is defined.Javascript variables have only two scopes:

  • Global Veriable - A global veriable has global scope which means it can be defined anywhere in your Javascript code.
  • Local Varibale - A local variable will be visible only within a function where it is defined. Function parameters are always local to that function.
